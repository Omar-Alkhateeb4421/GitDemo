## How To Work With Git.
1. git checkout main.
2. git pull.
3. git checkout -b bug-00-branch-name.
4. git add.
5. git commit.
6. git checkout main.
7. git pull.
8. git checkout bug-00-branch-name.
9. git merge main.
10. git checkout main.
11. git merge bug-00-branch-name.
12. git pull.
13. git push.
14. git branch -d bug-00-branch-name.
